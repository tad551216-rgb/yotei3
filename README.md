# わが家の予定板

家族3人(お父さん・お母さん・長男さん)の予定と日記を、iPhoneひとつで管理できる個人用アプリです。
サーバーやデータベースは不要で、すべてのデータは端末の中(ブラウザのlocalStorage / IndexedDB)にだけ保存されます。外部への送信は一切ありません。

## 機能

- **カレンダー** — 連続縦スクロールの万年カレンダー。長男さんの仕事の日は緑色で表示。日付タップでその日の一覧へ移動
- **一覧** — 縦に日付、横に3人を並べた連続スクロール表。お父さん・お母さんは登録済みリストからのタップ選択で予定を入力、長男さんはマスをタップするだけで「仕事⇔空白」を切り替え
- **日記** — 1日1ページの日記。写真の添付にも対応(自動縮小して端末内に保存)
- **設定** — 3人の名前の変更、予定内容リストの編集、データのバックアップ(書き出し/読み込み)

## 使い方

1. GitHub Pagesで公開されたURLをiPhoneのSafariで開く
2. 共有ボタン →「ホーム画面に追加」
3. 以後はホーム画面のアイコンからアプリのように起動できます(HTTPS配信のため、一度開けばオフラインでも起動します)

## データについて

- 予定・日記・写真はすべて閲覧している端末内に保存されます。このリポジトリやサーバーには何も保存されません
- Safariの「履歴とWebサイトデータを消去」を行うとデータも消えるため、設定画面のバックアップ機能で定期的に書き出しておくことをおすすめします

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JavaScriptすべて入り) |
| `sw.js` | オフライン起動用のService Worker |
| `manifest.json` | ホーム画面追加用の設定 |
| `icon-180.png` | ホーム画面用アイコン |
